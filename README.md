# UT Austin Contrast Grid

Test many foreground and background color combos for compliance with [WCAG 2.0 minimum contrast](https://www.w3.org/TR/UNDERSTANDING-WCAG20/visual-audio-contrast-contrast.html).

## How to Install and Run the Project

The project files are in `src`, so do your work there.

1. `npm install` to install all packages
2. `grunt dev` or `grunt` to run dev commands
3. `grunt build-dist` for building

Forked from [contrast-grid.eightshapes.com](http://contrast-grid.eightshapes.com).
